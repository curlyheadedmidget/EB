# About
```
The challenge.rb script will take a list of URLs and submit them to docraptor, creating pdfs for each URL. After the pdfs are created successfully, it will return some metadata for each of the urls ie. metadata, number of pages, info. The output will be in ascending order based on page count
```

# Setup
```
git clone git@github.com:curlyheadedmidget/EB.git
cd CodeChallenge
bundle install
```

# run script:
```
ruby challenge.rb
```
